# char-histo.pl

## Overview:
`char-histo.pl` is a script written in perl6 (raku).

It prints an overview of all used characters in a text file.

The output has the form of a sorted histogramm where the most
used character is at the top and the least used character at the bottom.


## Usage:
`./char-histo.pl <file>`

or

`perl6 char-histo.pl <file>`
